# 标签编辑器 - 模板分类说明

## 模板中心界面已实现

新的模板中心对话框已经实现，包含以下功能：

### 界面布局
- **顶部搜索栏**：支持按名称、尺寸等搜索模板
- **左侧分类栏**：显示所有可用的模板分类
- **右侧内容区**：
  - 新建空白页面：输入标签名称、宽度、高度
  - 模板展示页面：网格方式显示模板缩略图
- **底部按钮**：创建/取消按钮

### 模板分类列表

程序提供以下模板分类（已实现）：

1. **新建空白** - 创建自定义尺寸的空白标签
2. **我的模板** - 用户保存的自定义模板
3. **精选** (featured) - 精选优质模板
4. **基础** (basic) - 基础常用模板
5. **仓储物流** (warehouse) - 物流快递标签
6. **电子产品** (electronics) - 电子产品标签
7. **电商** (ecommerce) - 电商平台标签
8. **服饰** (clothing) - 服装鞋帽标签
9. **合格证** (certificate) - 产品合格证
10. **超市** (supermarket) - 超市零售标签
11. **小票** (receipt) - 收据小票模板
12. **人员证** (personnel) - 工作证、人员卡

### 模板文件存储位置

所有模板文件存储在应用数据目录下：

```
%APPDATA%/LabelEditor/templates/
├── my/              # 用户自定义模板
├── featured/        # 精选模板
├── basic/           # 基础模板
├── warehouse/       # 仓储物流模板
├── electronics/     # 电子产品模板
├── ecommerce/       # 电商模板
├── clothing/        # 服饰模板
├── certificate/     # 合格证模板
├── supermarket/     # 超市模板
├── receipt/         # 小票模板
└── personnel/       # 人员证模板
```

在Windows系统中，通常路径为：
`C:\Users\用户名\AppData\Roaming\LabelEditor\templates\`

### 如何添加模板

1. 在标签编辑器中设计好标签
2. 使用"文件" -> "保存"或"另存为"保存为 .lbl 文件
3. 将文件复制到对应的模板分类目录下
4. 重启应用或切换分类即可看到新模板

### 模板文件格式

模板文件使用 .lbl 扩展名，内部为 JSON 格式，包含：
- 标签尺寸信息
- 所有图形元素（文本、条码、二维码、图像、形状等）
- 元素属性和位置信息

### 功能特性

- ✅ 分类浏览：左侧列表切换不同分类
- ✅ 搜索功能：顶部搜索框快速查找模板
- ✅ 缩略图预览：网格显示模板缩略图
- ✅ 快速创建：双击模板直接创建
- ✅ 空白标签：支持自定义尺寸的空白标签
- ✅ 用户模板：保存和管理个人常用模板

### 使用流程

1. 点击"文件" -> "新建(N)"或工具栏的"新建"按钮
2. 在模板中心对话框中选择：
   - **新建空白**：输入尺寸创建空白标签
   - **选择模板**：从各分类中选择预设模板
3. 点击"创建"按钮或双击模板项
4. 开始编辑标签

### 后续工作

目前各个分类目录为空，需要逐步设计和添加模板文件到相应目录。

每个分类可以根据实际需求添加多个模板文件，文件名建议使用有意义的描述性名称，如：
- `物流标签_100x50.lbl`
- `FBA亚马逊标签_100x100.lbl`
- `价格签_竖版_60x40.lbl`
等

### 注意事项

- 模板文件必须是有效的 .lbl 格式（JSON）
- 建议为每个模板设置合适的尺寸和元素布局
- 可以随时添加或删除分类（需修改代码）
- 搜索功能支持中文模糊匹配
